# WFX AI ERP Frontend

React/Vite frontend for the WFX AI ERP Assistant.

This app will provide:

- ERP dashboard
- Natural-language AI assistant
- Product discovery search
- Image-based product discovery

## Local Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Default local URL:

```txt
http://localhost:5173
```

## Current Status

The frontend is scaffolded with Vite, React, Tailwind CSS configuration, and a
starter application shell. Feature pages will be added in later milestones.
